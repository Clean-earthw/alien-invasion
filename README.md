## üöÄ **Getting Started**

### **Prerequisites:**
- Meta Quest 3 or Meta Quest Pro headset
- Meta Horizon OS Browser (latest version)
- Development PC with Node.js 18+

### **Quick Start:**
```bash
# Clone the repository
git clone https://github.com/Clean-earthw/alien-invasion.git

# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

### **Deployment:**
The project is deployed and accessible at: `https://clean-earthw.github.io/alien-invasion/`
